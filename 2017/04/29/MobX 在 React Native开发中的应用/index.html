mobx标签

mobx常用的标签做一个解释。

  • @observable: 使用此标签监控要检测的数据；
  • @observer: 使用此标签监控当数据变化是要更新的Component（组件类）
  • @action:使用此标签监控数据改变的自定义方法(当在需要数据改变的时候执行此自定义方法，那么View层也会跟着自动变化，默认此View层已经使用@observer标签监控)
